# Museo Galleria del Premio Suzzara - Sito Web Ufficiale

Sito web ufficiale del Museo Galleria del Premio Suzzara - Il premio d'arte contemporanea più longevo d'Italia (dal 1948).

## 🎨 Contenuti

- **496 pagine HTML** complete e responsive
- **477 artisti** della collezione con schede dedicate
- **800+ opere** documentate dal 1948 ad oggi
- Design moderno con design system completo
- SEO ottimizzato con redirect 301

## 🏗️ Struttura

```
/
├── index.html                    # Homepage
├── museo/                        # Sezione museo
├── premio/                       # Sezione premio
├── collezione/                   # Collezione (artisti + opere)
│   └── artisti/                 # 477 pagine artisti
├── mostre/                       # Mostre temporanee
├── eventi/                       # Eventi e attività
├── visita/                       # Info pratiche
├── contatti/                     # Contatti
├── didattica/                    # Didattica museale
├── esponi/                       # Esponi con noi
├── ura/                          # Progetto URA
├── edizionidelpremio/           # Edizioni del Premio
└── _redirects                    # Redirect 301 SEO
```

## 🚀 Deploy

Il sito è deployato automaticamente su **Render** da questo repository GitHub.

### Auto-deploy workflow:
1. Modifica file localmente o su GitHub
2. Commit e push su branch `main`
3. Render fa auto-deploy in ~30 secondi
4. Sito aggiornato online
